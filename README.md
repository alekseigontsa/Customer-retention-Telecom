# Customer-retention-Telecom

## Удержание клиентов Телеком

Cтоит задача проанализировать данные и спрогнозировать отток пользователей (выявить людей, которые продлят контракт и которые не продлят).

### Исследование состоит из четырех частей.

1. Опсание данных. 

Загрузка данных.
Проверка на корректность и пропуски. Преобразование данных для удобства исследования.

2. Исследование зависимостей.

Проведен раздочный анализ. Просмотрены зависимости между параметрами. Распределение параметров.

3. Модели для прогнозирования оттока.

Данные были разделены на тренировочные и тестовые выборки.
Для прогнозирования оттока клиентов были проверены модели. Для определения качества модели использовали метрику "roc_auc".

- KNeighborsClassifier
- GaussianNB
- DecisionTreeClassifier
- LogisticRegression
- SGDClassifier
- SVC
- AdaBoostClassifier
- RandomForestClassifier
- GradientBoostingClassifier

4. Сравнение качества моделей.

Проведено сравнение метрики моделей и определена модель, которая имеет набольшее значение по метрике 'roc_auc'.
